# Выбор типа вычислений CTranslate2

## Цель

Запускать распознавание на CUDA без ошибки, если видеокарта не поддерживает `float16`.

## Решение

Сервис расшифровки выбирает тип вычислений из поддерживаемых CTranslate2 режимов. Для CUDA порядок предпочтения: `float16`, `int8_float16`, `int8_float32`, `int8`, `float32`. Для CPU сохраняется `int8`.

## Ошибки и проверка

Если CTranslate2 не предоставляет подходящий CUDA-режим, сервис продолжит использовать `float32`; существующая русская ошибка и запись traceback в `logs/debug.log` сохраняются. Unit-тест закрепляет выбор `int8_float32` для набора режимов GTX 1080 Ti.
