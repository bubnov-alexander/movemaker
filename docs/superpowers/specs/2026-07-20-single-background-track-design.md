# Один необязательный путь фоновой музыки

## Цель

Разрешить конфигурацию фоновой музыки с одним из путей `epic_path` или `calm_path`, не требуя оба файла.

## Поведение

- Если указаны оба пути, выбор трека по сцене не меняется.
- Если указан только `epic_path`, он используется как для эпичных, так и для спокойных сцен.
- Если указан только `calm_path`, он используется как для спокойных, так и для эпичных сцен.
- Если не указан ни один путь, конфигурация завершается русскоязычной ошибкой.
- Существующие конфигурации с двумя путями остаются совместимыми.

## Конфигурация

Допустимая конфигурация с одним треком:

```yaml
background_music:
  calm_path: music/altyn.mp3
  max_volume: 0.12
  quiet_volume: 0.18
```

Локальный `config.yaml` использует этот вариант и безопасные уровни громкости, чтобы исходная речь оставалась слышна.

## Проверка

- Unit-тесты загрузки YAML проверяют конфигурации только с `epic_path` и только с `calm_path`.
- Unit-тест выбора трека проверяет, что один указанный файл выбирается для сцен обоих типов.
- Полный тестовый набор остаётся зелёным.
