# Зеркалирование видеоряда

## Цель

Горизонтально зеркалить весь видеоряд в итоговом Shorts, не изменяя читаемость субтитров.

## Поведение

- Базовый размытый слой исходного видео зеркалится.
- Верхнее зацикленное видео из `layout_background_path` зеркалится.
- Основной фрагмент в средней зоне зеркалится.
- ASS-субтитры накладываются после преобразований видео и не зеркалятся.
- Аудиологика не меняется.

## Реализация и проверка

FFmpeg-граф добавляет `hflip` к каждому видеослою перед наложением. Unit-тест команды проверяет наличие `hflip` и порядок наложения субтитров после видеокомпозиции. Интеграционный тест подтверждает создание MP4.
